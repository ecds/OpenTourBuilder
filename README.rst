Battle of Atlanta
=================
-----------------
[![Build Status](https://travis-ci.org/emory-libraries-disc/battleofatlanta.png?branch=develop)](https://travis-ci.org/emory-libraries-disc/battleofatlanta)

Overview
--------
A Django project that allows creation of tours for the Battle of Atlanta project. \
The tours are targeted towards mobile phones and designed to be used while exploring \
tour stops.
